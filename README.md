## 📖 概要

ソフトウェア開発は、HTML, CSS, JavaScriptを活用したおみくじアプリケーションです。

### 主な機能
- 3×3 マスの対戦（プレイヤー vs プレイヤー）
- 勝敗・引き分け判定
- リスタート機能（盤面リセット）
- シンプルなUI（学習用）

### 技術スタック
- **フロントエンド**: HTML / CSS / JavaScript / BootStrap  
- **開発環境**: Visual Studio Code
- **使用ツール**: Git / Live Server
