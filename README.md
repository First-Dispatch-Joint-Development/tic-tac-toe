# Tic-Tac-Toe（三目並べ）

HTML / CSS / JavaScript を用いて実装した三目並べアプリケーションです。  
5人チームでの共同開発を通じて、GitHub運用や役割分担を学ぶことを目的としています。

---

## 主な機能
- 3×3 マスの対戦（プレイヤー vs プレイヤー）
- 勝敗・引き分け判定
- リスタート機能（盤面リセット）
- シンプルなUI（学習用）

---

## 技術スタック
- **フロントエンド**: HTML / CSS / JavaScript / BootStrap  
- **開発環境**: Visual Studio Code, Live Server
